1. Электрический заряд. Закон сохранения электрического заряда. Закон Кулона.
	1. Заряды бывают двух видов: положительные и отрицательные
	   Заряд дискретен, т.е. $q = ne$
	2. Закон сохранения заряда
	   Сумма зарядов в изолированной системе не изменяется при любых процессах, происходящих в этой системе
	3. Закон Кулона
	   Сила взаимодействия между зарядами прямо пропорциональна взаимодействующим зарядам и обратно пропорциональна квадрату расстояния между ними$$F_{Кл} = \frac{1}{4 \pi \epsilon_{0}} \frac{q_{1}q_{2}}{r^2}$$
2. Электростатическое поле в вакууме. Напряженность поля. Принцип суперпозиции полей.
	1. Электростатическое поле - особый вид материи, возникающий вокруг статичных зарядов. Заряды взаимодействуют друг с другом через это поле. 
	2. Напряжённость поля - силовая характеристика поля $E = \frac{F}{q_{0}}$
	3. Принцип суперпозиции
	   Результирующий вектор напряжённости поля равен векторной сумме напряжённостей, создаваемых всеми зарядами$$\vec{E} = \sum \vec{E_{i}}$$
3. Поле диполя. Диполь во внешнем поле.
	1. Диполь - система, состоящая из двух зарядов, равных по модулю и противоположных по знаку и находящихся на расстоянии $l$ друг от друга
	   Главная характеристика диполя - дипольный момент $\vec{p} = ql$
	2. При помещении во внешнее однородное поле на оба заряда диполя действуют силы, равные по модулю и противоположные по направлению, следовательно, результирующая сила, действующая на диполь во внешнем однородном поле, равна нулю
	3. При помещении диполя во внешнее однородное поле в диполе возникает момент силы, стремящейся повернуть диполь так, чтобы вектор дипольного момента стал сонаправлен с вектором напряжённости поля
	   Этот момент силы числено равен векторному произведению вектора дипольного момента на вектор напряжённости поля$$M = [\vec{p} \times \vec{E}]$$
4. Поток вектора напряженности. Теорема Гаусса для электростатического поля в вакууме.
	1. Поток вектора напряжённости$$\Phi = \oint_{S} EdS$$
	   При прохождении через плоскую поверхность $\Phi = 2ES$
	2. Теорема Гаусса для электростатического поля в вакууме
	   Если линии напряжённости проходят через замкнутую плоскость, то поток вектора напряжённости равен сумме зарядов, лежащих в этой плоскости, делённой на электрическую постоянную$$\Phi = \oint_{S} EdS = \frac{\sum{q_{i}}}{\epsilon_{0}}$$
5. Вычисления напряженности поля электростатических систем по теореме Гаусса
	1. Бесконечная заряженная плоскость
	   Рассмотрим участок плоскости, равный сечению цилиндра(площадь сечения - S)
	   Тогда по теореме Гаусса $\Phi = \oint_{S} EdS =2ES = \frac{q}{\epsilon_{0}}$
	   Введём величину $\sigma$ - поверхностная плотность заряда, $\sigma = \frac{q}{S} \implies q = S \sigma$
	   $\Phi = 2ES = \frac{q}{\epsilon_{0}} = \frac{\sigma S}{\epsilon_{0}} \implies E = \frac{\sigma}{2 \epsilon}$
	2. Система двух разнозаряженных бесконечных плоскостей
		1. Снаружи:
		   $E = 0$, т.к. вектора напряжённости полей, создаваемых отрицательно заряжённой пластиной и положительно заряженной пластиной равны по модулю и противоположны по направлению
		2. Внутри: 
		   $\vec{E} = \vec{E_{+}} + \vec{E_{-}} = \frac{\sigma}{2 \epsilon} + \frac{\sigma}{2 \epsilon} = \frac{\sigma}{\epsilon}$
	3. Заряженная сфера
		1. Снаружи: $E = 0$
		2. Внутри: $E = \frac{q}{4 \pi \epsilon_{0} r^2}$
 6. Работа сил электростатического поля. Потенциал. Теорема о циркуляции вектора напряженности.
	 1. Работа сил электростатического поля $dA = Fdr \implies A = \int_{r_{1}}^{r_{2}} F \, dr = \frac{qq_{0}}{4 \pi \epsilon_{0}}\left( \frac{1}{r_{1}} - \frac{1}{r_{2}} \right)$
	 2. Потенциал - потенциальная энергия заряд в данной точке поля$$\phi = \frac{W_{пт}}{q_{0}}$$
	 3. Теорема о циркуляции вектора напряжённости
	    Если траектория движения заряда под действием внешнего поля - замкнутый контур, то работа сил внешнего поля по перемещению этого заряда равна 0(т.к. $r_{1} = r_{2}$)
	    Следовательно, $\oint_{l} Edl = 0$
	    Следовательно, линии напряжённости электрического поля не могут быть замкнутыми и могу начинаться, либо заканчиваться только на зарядах
 7. Связь между напряженностью и потенциалом.
    $\vec{E} = -grad \phi$
 8. Поляризация диэлектриков. Напряженность поля в диэлектриках.
	 1. Диэлектрики - вещества, не обладающие свободными зарядами, следовательно, не проводящие ток
	 2. Виды диэлектриков:
		 1. Неполярные - молекулы которых симметричны
		 2. Полярные - молекулы которых несимметричны
		 3. Ионные - у которых есть 2 подрешётки
	3. Виды поляризации:
		1. Электронная - дипольный момент возникает защеку деформации электронных оболочек
		2. Ориентационная - дипольный момент возникает за счёт вращения молекулы
		3. Ионная - дипольный момент возникает защеку смещения подрешёток относительно друг друга
		4. Напряжённость поля в диэлектриках - 💀

 10. Проводники в электростатическом поле.
	 1. При помещении проводника в электрическое поле разность потенциалов внутри проводника стремиться выровняться, а поле внутри проводника стремится к нулю
	 2. Поверхность проводника эквипотенциальна
	 3. Заряды, переходящие внутри проводника называются индуцированными, а явление перераспределения зарядов внутри проводника называется электростатической индукцией
 11. Конденсаторы. Электроемкость конденсаторов. Соединения конденсаторов.
	 1. Конденсатор - устройство, состоящее из двух разнозаряженных проводников
	 2. Электроёмкость конденсатора - коэффициент пропорцональности между зарядом и разностью потенциалов($C = \frac{q}{\phi}$)
		 1. Плоский конденсатор
		    $C = \frac{\epsilon\epsilon_{0}S}{d}$
		 2. Сферический конденсатор
		    $C = 4 \pi \epsilon \epsilon_{0} \frac{r_{1}r_{2}}{r_{1}-r_{2}}$
	3. Соединения конденсаторов
		1. Параллельное
		   $C = \sum C_{i}$
		2. Последовательное
		   $\frac{1}{C} = \sum \frac{1}{C_{i}}$
 12. Энергия взаимодействия системы зарядов. Энергия конденсатора и заряженного проводника. Энергия электрического поля.
	 1. Энергия взаимодействия системы зарядов
	    $W = \sum q_{i} \phi_{i}$
	 2. Конденсатора
	    $W = \frac{qU}{2} = \frac{CU^2}{2} = \frac{q^2}{2C}$
	 3. Заряженного проводника
	    $W = \frac{q\phi}{2}$
	 4. Энергия электрического поля
	    $w = \frac{W}{V} = \frac{\epsilon\epsilon_{0}E^2}{2}$
13. Электрический ток. Сила тока и плотность тока.
	1. Электрический ток - упорядоченное движение заряженных частиц
	   За направление тока принято брать направление движения положительных зарядов
	2. Сила тока - характеристика тока, показывающая количество заряда, прошедшего через проводник за отрезок времени $I = \frac{dq}{dt}$
	   При постоянном токе $I = \frac{q}{t}$, то есть сила тока и заряд - постоянные
	3. Для исследования постоянного тока была введена микроскопическая величина - плотность тока - показывающая количество заряда, прошедшего через проводник поперечным сечением S за время t$$\vec{j} = \frac{q}{S_{\perp}t} = \frac{I}{S_{\perp}}$$
14. ЭДС.
	1. Электродвижущая сила - суммарная работа сторонних(неэлектрических) сил, перемещающих заряд
	   $\epsilon = \frac{A_{ст}}{q_{0}}$
	2. На однородном участке цепи$$U_{12} = \phi_{1} - \phi_{2}$$
	3. На неоднородном участке цепи$$U_{12} = (\phi_{1} - \phi_{2}) + \epsilon_{12}$$

19. Разветвленные цепи. Правила Кирхгофа.
	1. Узел - точка соединения 3 и более участков цепи
	2. 1 правило Кирхгофа
	   В узлах разветвлённой цепи алгебраическая сумма сил тока равна 0$$\sum I_{k} = 0$$
	3. 2 правило Кирхгофа
	   В замкнутых контурах в разветвлённой цепи сумма падений тока равна сумме ЭДС$$\sum I_{k} R_{k} = \sum \epsilon_{k}$$
	4. Правила Кирхгофа позволяют узнать ток на любом участке разветвлённой цепи по следующему алгоритму
		1. Произвольно выбрать направление тока на всех участках цепи, если в конце вычислений $I>0$, то выбранное направление совпадает с реальным
		2. Найти все узлы в цепи и составить для каждого уравнение по 1 правилу Кирхгофа
		3. Найти все замкнутые контуры в цепи и составить для каждого уравнение по 2 правилу Кирхгофа
		4. Проверить, что сопротивления и ЭДС всех участков цепи есть в системе уравнений
		5. Решить систему уравнений
