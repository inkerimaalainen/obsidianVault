21. Основные понятия: фаза, компонент, число степеней свободы. Правило фаз Гиббса.
	1. Фаза - однородные части различающиеся по физическим свойствам, отделённый друг от друга границами раздела
	2. Компонент - независимое индивидуальное вещество, которое является составной частью системы и может быть выделено из неё и существовать самостоятельно
	3. Число степеней свободы(вариантность системы) - число термодинамических параметров, которые могут изменяться независимо друг от друга, без нарушения равновесия системы
	   Число степеней свободы системы зависит от числа компонентов, числа равновесных фаз и количества внешних прееменных, эта зависимость называется ==правилом фаз Гиббса==
	4. Правило фаз Гиббса: $$C = K - Ф + n$$, где С(или f) - число степеней свободы, К - число компонентов в системе, n - число внешних изменяющихся параметров, Ф - число фаз в системе
22. Принципы, правила и законы, используемые при построении диаграмм состояния
	1. На диаграмме откладывают на оси абсцисс(x) температуру, а на оси ординат(y) - давление
	2. На диаграмме имеется 3 поля, отвечающие 3 агрегатным состояниям
	3. В пределах каждого поля число степеней свободы будет равно 2
	4. Линии ограничиваю поля этих фаз, показывают те значения температур и давлений при который в системе имеется равновесие между 2 данными фазами
	5. В каждой точке, лежащей на 1 такой кривой количество степеней свободы равно й
	6. В точке пересечения всех 3 кривых количество степеней свободы равно 0
