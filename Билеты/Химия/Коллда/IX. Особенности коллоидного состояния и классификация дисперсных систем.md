56. **Особенности коллоидного состояния: гетерогенность, дисперсность, термодинамическая неустойчивость и кинетическая устойчивость**
	1. ==Дисперсные системы== - это двух-, или, в общем случае, многофазные ==гетерогенные== системы, в которых хотя бы одна из фаз представлена малыми частицами, размеры которых, однако, превосходят молекулярные$$D = \frac{1}{l} \quad n = \frac{l_{исх}}{l_{кон}}$$
		   $D$ - дисперсность(раздробленность) вещества, $l$ - линейный размер частиц, $n$ - степень дисперсии
	2. ==Гетерогенность== - качественная характеристика дисперсных систем
	3. ==Дисперсность== - количественная характеристика дисперсных систем
		1. ==Удельная поверхность== - мера дисперсности, суммарная поверхность всех частиц в единице объёма или массы вещества$$S_{уд} = \frac{S}{V} = [м^{-1}] = \frac{S}{m} = \left[ \frac{м^2}{кг} \right]$$, где $S$ - общая поверхность между фазами, $V$ - объём дисперсионной фазы, $ρ$ - плотность дисперсной фазы
57. **Принципы классификации дисперсных систем и их классификация: по агрегатному состоянию, по дисперсности, по внутреннему взаимодействию, по межфазному взаимодействию.**
	1. По размерам частиц(дисперсности)
		1. Высокодисперсные(ультрамикрогетерогенные)
		2. Среднедисперсные
		3. Грубодисперсные![[Pasted image 20260413124058.png]]
	2. По форме дисперсных частиц
		1. Трёхмерные
			1. Для куба: $$S = 6l^2 \quad V = l^3 \implies S_{уд} = \frac{S}{V} =  \frac{N6l^2}{Nl^3} = \frac{6}{l}$$
			2. Для шара: $$S = 4\pi r^2 \quad V = \frac{4}{3} \pi r^3 \implies S_{уд} = \frac{S}{V} = \frac{4\pi r^2}{\frac{4}{3}\pi r^3} = \frac{3}{r} = \frac{6}{l}$$
			3. Таким образом в общем случае:$$S_{уд} = \frac{6}{l} = 6D = \frac{6}{lρ}$$
		2. Двухмерные$$S = \pi lL \quad V = \frac{\pi l^2}{4}L \implies S = \frac{4}{l} = 4D$$
		3. Одномерные$$S = 2Lh \quad V = Lhl \implies S_{уд} = \frac{2}{l} = 2D$$
		4. В общем случае: $S_{уд} = KD$
			1. Для трёхмерных систем $K = 6$
			2. Для двухмерных $K = 4$
			3. Для одномерных $K = 2$
	3. По соотношению размеров частиц дисперсной фазы
		1. Монодисперсные системы - системы, состоящие из частиц одного размера
		2. Полидисперсные системы - системы состоящие из частиц разных размеров
	4. По характеру межфазного взаимодействия(термодинамический устойчивости)
		1. Лиофильные(термодинамически устойчивые) - с сильным межмолекулярным взаимодействием
		2. Лиофобные(термодинамически неустойчивые) - со слабым межмолекулярным взаимодействием 
	5. По кинетическим свойствам дисперсной фазы
		1. Свободнодисперсные - частицы дисперсной фазы не связаны между собой и могут свободно передвигаться в дисперсионной среде
		2. Связнодисперсные системы - частицы связаны друг с другом, оразуя пространственные сетки
