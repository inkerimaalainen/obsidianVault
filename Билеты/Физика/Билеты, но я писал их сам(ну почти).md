1. Кинематика. Основные понятия и определения.
	1. Кинематика - раздел механики, изучающий движение тел, независимо от его причин
	2. Основные понятия и определения в кинематике
		1. Материальная точка - тело, размерами и формой которого в данной задаче можно пренебречь
		2. Система отсчёта - абсолютно твёрдое тело, жёстко связанное с началом системы координат и снабжённое часами, относительно которого считается положение тела в пространстве и время
		3. Радиус-вектор($\vec{r}$) - вектор, соединяющий начало координат и точку, в которой находится тело в данный момент
		4. Траектория движения - сплошная линия, вдоль которой движется тело на протяжении всего своего движения
		5. Путь($S$) - длина траектории движения тела между его начальным и конечным положениями
		6. Перемещение($\Delta x$, $\Delta \vec{r}$) - вектор, соединяющий точки начального и конечного положения тела
		7. Скорость - физическая величина, показывающее изменение положения тела в пространстве с течением времени$$\vec{v} = \frac{d\vec{r}}{dt}$$Также$$|\vec{v}| = v = \sqrt{ v^2_{x} + v^2_{y} + v^2_{z} }$$
		8. Средняя скорость$$<v> = \frac{dS}{dt}$$
		9. Ускорение$$\vec{a} = \frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$$
		10. Среднее ускорение$$<\vec{a}> = \frac{\Delta \vec{v}}{\Delta t}$$
		11. Моментальное ускорение$$\lim_{ \Delta t \to 0 } <\vec{a}> = \lim_{ \Delta t \to 0 } \frac{\Delta \vec{v}}{\Delta t} = \frac{d\vec{v}}{dt} = \vec{v}'$$
2. Кинематика поступательного движения. Ускорение при криволинейном движении.
	1. При криволинейном движении ускорение делится на 2 вида
		1. Тангенциальное - направлено по касательной к траектории движения и характеризует изменение скорости тела с течением времени$$a_{t} = \frac{d\vec{v}}{dt} = \epsilon R$$
		2. Нормальное - направлено перпендикулярно вектору моментальной скорости и характеризует изменение траектории движения с течением времени$$a_{n} = \frac{v^2}{R} = \omega^2R$$
		3. Полное ускорение при криволинейном движении$$a = \sqrt{ a^2_{n} + a^2_{t} }$$
3. Кинематика вращательного движения.
	1. Угол поворота($\phi$) - угол между радиус-векторами тела в начальном и конечном его положении
	2. Угловая скорость$$\omega = \frac{d\phi}{dt}$$
	3. Угловое ускорение$$\epsilon = \frac{d\omega}{dt} = \frac{d^2\omega}{dt^2}$$
	4. Вращательное движение - частный случай криволинейного движения, то есть моментальное ускорение так же делится на нормальное и тангенциальное
4. Законы Ньютона. Масса, импульс, сила.
	1. Законы Ньютона
		1. Существуют такие системы отсчёта, в которых тело покоится или движется равномерно только тогда, когда векторная сумма сил, действующих на него, равна 0. Такие системы отсчёта называются инерциальными
		2. Сила придаёт телу ускорение$$\vec{F} = m\vec{a} = \frac{dm\vec{v}}{dt} = \frac{d\vec{p}}{dt}$$
		3. Действие тел друг на друга носит характер взаимодействия. Если одно тело действует на другое с силой $\vec{F_{12}}$, то на это тело начнёт действовать сила $\vec{F_{21}}$, где $$\vec{F_{12}} = -\vec{F_{21}}$$
	2. Масса($m$) - мера инертности тела
	3. Импульс($\vec{p}$) - физическая величина, числено равная произведению скорости тела на его массу$$\vec{p} = m\vec{v}$$
	4. Сила - мера взаимодействия тел друг с другом, физическая причина изменения движения тел
5. Центр масс системы тел и закон его движения.
	1. Центр массы тела - точка приложения вектора результирующей силы, действующей на тело
		1. Радиус-вектор центра массы тела$$\vec{r_{c}} = \frac{1}{m} \sum m_{i}\vec{r}_{i}$$где $m = \sum m_{i}$
		2. Скорость центра массы тела$$\vec{v_{c}} = \frac{d\vec{r_{c}}}{dt} = \frac{d}{dt} \frac{1}{m} \sum m_{i}\vec{r_{i}} = \frac{1}{m} \sum \frac{dm_{i}\vec{r_{i}}}{dt} = \frac{1}{m} \sum m_{i}\vec{v_{i}} \implies m\vec{v_{c}} = \sum m_{i}\vec{v}_{i}$$
		3. $$\vec{p_{c}} = \sum p_{i} = \sum m_{i}\vec{v_{i}} = m\vec{v_{c}}$$
	2. Закон движения центра массы тела$$\vec{F_{внеш}} = \frac{d\vec{p_{c}}}{dt} = m \frac{d\vec{v_{c}}}{dt} = m\vec{a_{c}}$$Сравним с выражением для i-точки$$\vec{F_{i}} = \frac{d\vec{p_{i}}}{dt} = m \frac{d\vec{v_{i}}}{dt} = m\vec{a_{i}}$$Выражения одинаковы, следовательно, движение всего тела можно описать движением материальной точки, находящейся в его центре масс
	3. Центр тяжести тела - точка, через которую проходит равнодействующая сил тяжести всех точек тела
6. Система материальных точек. Закон сохранения и изменения импульса.
	1. Система материальных точек - совокупность конечного количества материальных точек, взаимное расположение которых может изменятся с течением времени
	2. Закон сохранения импульса
	   Импульс замкнутой системы материальных точек не изменяется при любых процессах, происходящих внутри системы
	3. Закон изменения импульса$$\vec{F_{i}} = \frac{d\vec{p_{i}}}{dt}$$$$F_{i} = F_{i \text{ внеш}} + F_{i \text{ внутр}}$$Изменение импульса системы равно результирующей внешней силе, приложенной к ней
7. Энергия, работа, мощность. Кинетическая и потенциальная энергия тел. Связь между потенциальной энергией и силой.
	1. Энергия - физическая величина, характеризующая способность тела совершать работу
		1. Кинетическая - характеризует энергию движения тела$$E_{k} = \frac{m_{i}v^2_{i}}{2}$$$$dE_{k} = dA$$
	2. Потенциальная - характеризует энергию взаимодействия тел
	   Конкретный вид функции $E_{п}$ определяется видом силы $F$$$\text{В поле силы тяжести: }E_{п} = mgh + E_{п0}$$$$\text{В гравитационном поле: } E_{п} = E_{п0} - G \frac{mM}{R + h}$$$$\text{Потенциальная энергия упругой деформации: } E_{п} = kx^2$$
	3. Работа$$d\vec{A} = \vec{F}d\vec{r} = Fdr\cos \alpha$$$$dS = |d\vec{r}| = dr \implies d\vec{A} = \vec{F}dS$$
	4. Мощность$$P = \frac{dA}{dt} = \frac{\vec{F}d\vec{r}}{dt} = \vec{F}\vec{v}\text{, где }\vec{v}\text{ - скорость точки приложения силы}$$
8. Потенциальные и непотенциальные силы. Закон сохранения и изменения энергии в механике.
	1. Потенциальные силы - силы, работа которых зависит только от начального и конечного положений тела и не зависит от траектории его движения
	2. Непотенциальные силы - силы, работа которых зависит от вида траектории движения тела
	3. Закон сохранения энергии
	   Механическая энергия системы постоянна, если на неё действуют только потенциальные силы
	4. Закон изменения энергии$$E = E_{k} + E_{п} + E_{вз} = \frac{m_{i}v_{i}^2}{2} + \sum E_{пi} + E_{вз}$$$$dE = d(\frac{m_{i}v_{i}^2}{2} + \sum E_{пi} + E_{вз}) = dA_{непот} = dA^{внеш}_{непот} + dA^{внутр}_{непот}$$
9. Момент силы, момент импульса, момент инерции. Основное уравнение динамики вращательного движения.
	1. Момент силы - вектор, равный векторному произведению радиус-вектора на вектор силы$$M = [\vec{r}\vec{F}]$$
	2. Момент импульса - вектор, равный векторному произведению радиус-вектора на вектор импульса тела$$L = [\vec{r}\vec{p}] = \vec{r}m\vec{v}$$
	3. Момент инерции - мера инертности тела при вращательном движении
	4. Основное уравнение динамики вращательного движения$$M = I\epsilon$$
10. Момент инерции и примеры его вычисления.
	1. Относительно неподвижной оси: $I = \sum_{i}m_{i}\vec{r}^2_{i}$ 
	2. Для стержня: $I = \frac{ml^2}{12}$, где $l$ - длина стержня относительно оси, проходящей перпендикулярно ему через его центр
	3. Для шара: $I = \frac{2}{5}mR^2$
	4. Для цилиндра: $I = \frac{mR^2}{2}$
	5. Для кольца: $I = mR^2$
11. Теорема Штейнера и ее применение для вычисления моментов инерции тел.
	1. Момент инерции относительно произвольной оси равен сумме момента инерции относительно оси, параллельной данной и проходящей через центр масс тела и произведения массы тела на квадрат расстояния между осями$$I = I_{c} + md^2$$
	2. Применение для вычисления моментов инерции
	   Для вращения однородного стержня вокруг его конца$$I = I_{c} + md^2 = \frac{ml^2}{12} + \frac{ml^2}{4} = \frac{ml^2}{3}$$
12. Закон сохранения момента импульса.
    Если внешние силы отсутствуют или их векторная сумма равна 0, то$$I\epsilon = I \frac{d\omega}{dt} = \frac{d}{dt}I\omega = 0 \implies I\omega = const$$
13. Кинетическая энергия вращающегося тела. Работа внешней силы при вращении тела.
	1. $E_{kвращi} = \frac{\Delta m_{i} v_{i}^2}{2} = \frac{\Delta m_{i}R_{i}^2}{2}\omega^2$ 
	   $E_{kвращ} = \sum E_{kвращi} = \sum \frac{\Delta m_{i}R_{i}^2}{2}\omega^2 = \frac{I\omega^2}{2}$
	2. $dA = dE_{k} = \frac{dI\omega^2}{2} =I\omega \epsilon dt = I \epsilon d\phi = M_{внеш}d\phi$
14. Предмет молекулярной физики. Тепловое движение. Идеальный газ. Параметры состояния идеального газа.
	1. Молекулярная физика - раздел физики, изучающий физические свойства тел в зависимости от их строения, силами взаимодействия между образующими их молекулами, атомами и ионами и характер их теплового движения
	2. Молекулярно-кинетическая теория
	   Все тела состоят из мельчайших частиц, которые взаимодействуют друг с другом и находятся в постоянном беспорядочном тепловом движении
	3. Термодинамическая система - совокупность рассматриваемых тел, которые могут обмениваться энергией друг с другом и со внешней средой
	4. Термодинамические параметры системы
		1. Давление - сила, действующая на единицу поверхности
		2. Температура - понятие, имеющее смысл только в равновесных состояниях системы
	5. Равновесное состояние системы - такое состояние системы, при котором все термодинамические параметры системы остаются неизменными
	6. Процесс - переход из одного состояния в другое
	7. Процесс можно назвать равновесным, если его можно представить как непрерывную последовательность равновесных состояний системы
	8. Идеальный газ - газ, молекулы которого не взаимодействуют друг с другом
	   Для идеального газа уравнение состояния имеет вид$$\frac{pV}{T} = const$$
15.  Изопроцессы в газах. Уравнение Менделеева – Клапейрона.
	1. Изопроцессы
		1. Изохорный - закон Шарля$$V = const \text{ - изохора; } \frac{p}{T} = const$$
		2. Изобарный - закон Гей-Люссака$$p = const \text{ - изобара; } \frac{V}{T} = const$$
		3. Изотермический - закон Бойля-Мариотта$$T = const \text{ - изотерма; } pV = const$$
	2. Уравнение Менделеева-Клапейрона$$pV = \frac{m}{M}RT = \nu RT$$$$n = \frac{N}{V} \text{ - концентрация молекул}$$$$p = \frac{\nu RT}{V} = \frac{N}{N_{A}} N_{A} \frac{R}{N_{A}} \frac{T}{V} = N \frac{R}{N_{A}} \frac{T}{V} = \frac{nRT}{N_{A}} = nkT $$где $k$ - постоянная больцмана($k = \frac{R}{N_{A}}$)
16.  Основное уравнение молекулярно – кинетической теории газов и его следствие. Средняя кинетическая энергия молекул.
	1. Предположим, что скорости всех молекул одинаковы и равны $v$. Тогда до участка $\Delta S$ за время $\Delta t$ долетят молекулы, занимающие объём параллелепипеда с основанием $\Delta S$ и высотой $v\Delta t$. Если концентрация молекул равна $n$, то в параллелепипеде их будет $\Delta N = nv\Delta t\Delta S$, а со стенкой столкнётся $\Delta N = \frac{1}{6}nv\Delta t\Delta S$
	   Удары о стенки сосуда абсолютно упругие, следовательно, изменение импульса после удара будет равным $-2mv$, тогда приобретённый стенкой импульс по закону сохранения импульса будет равен $2mv$
	   Тогда общий импульс, который получит стенка будет равен $\frac{1}{3}nv^2m\Delta t\Delta S$
	   Импульс силы - физическая величина, характеризующая воздействие силы на тело в течение некоторого времени $I = F \Delta t$
	   Тогда $\frac{1}{3}nv^2m\Delta t\Delta S = p\Delta S\Delta t$
	2. Если учесть, что молекулы движутся с разными скоростями, то надо заменить $nv^2$ на $n\sum_{i=1}^n \frac{v^2}{n} = n<v^2>$ 
	   Тогда $p = \frac{1}{3}mn<v^2>$
	   $\sqrt{ <v^2> }$ - среднеквадратичная скорость молекул
	3. $p = \frac{2}{3}n \frac{m<v^2>}{2} = \frac{2}{3}n<E_{пост}>$ - основное уравнение МКТ(уравнение Клазиуса), где $<E_{пост}>$ - средняя кинетическая энергия поступательного движения молекул
	4. $p = nkT = \frac{2}{3}n<E_{пост}> \implies <E_{пост}> = \frac{3}{2}kT$
17. Закон Максвелла для распределения молекул идеального газа по скоростям.
	1. Закон Максвелла$$F(v) = 4\pi\left( \frac{m}{2 \pi kT} \right)^{\frac{3}{2}} \exp\left( -\frac{mv^2}{2kT} \right)v^2$$где $N$ - общее число молекул, 
	   $m$ - масса одной молекулы
	2. Максимальная скорость1$$v = \sqrt{ \frac{2kT}{m} } = \sqrt{ \frac{2RT}{M} }$$
	3. Наиболее вероятная скорость$$v_{max} = \sqrt{ \frac{8kT}{\pi m} } = \sqrt{ \frac{8RT}{\pi M} }$$
	4. Среднеквадратичная скорость$$\sqrt{ <v^2> } = \sqrt{ \frac{3kT}{m} } = \sqrt{ \frac{3RT}{M} }$$

20. Число столкновений и средняя длина свободного пробега молекул.
	1. Длина свободного пробега - расстояние, которое молекула проходит между столкновениями($\lambda_{i}$)
	2. Среднее число столкновений  единицу времени$$<z> = n\pi d^2<v>$$где $d$ - эффективный диаметр, т.е. наименьшее расстояние, на которое сближаются центры молекул при столкновении
	3. Число столкновений в единицу времени$$z = \sqrt{ 2 }n\pi d^2<v>$$
	4. Средняя длина свободного пробега$$<\lambda> = \frac{1}{z} = \frac{1}{\sqrt{ 2 }n\pi d^2}$$
21.  Явления переноса в газах. Опытные законы диффузии, теплопроводности и внутреннего трения.
	1. Диффузия - явление уравнивания концентрации вещества в системе вследствие теплового движения
	   Опытный закон - закон Фика(диффузионный поток)$$N_{i} = -D \frac{dn}{dx}$$где $D$ - коэффициент диффузии
	2. Теплопроводность - явление уравнивания температуры в системе
	   Опытный закон - закон Фурье(поток теплопроводности)$$Q = -\chi  \frac{dT}{dx}$$где $\chi$ - коэффициент теплопроводности
	3. Вязкость
	   Опытный закон - закон Ньютона для внутреннего трения$$F = \eta |\frac{dU}{dx}|S$$где $\eta$ - коэффициент вязкости
	   $S$ - площадь соприкасающихся слоёв
	   $\frac{dU}{sx}$ - градиент скорости
22.  Предмет термодинамики. Теплота и работа как формы передачи энергии.
	1. Термодинамика - раздел физики, изучающий различные свойства тел и изменения состояния вещества, не учитывая внутреннее строение тел и характер движения отдельных частиц, а основанный на изучении превращения энергии и происходящего в системе
	2. Термодинамическая система - совокупность рассматриваемых тел, которые могут обмениваться энергией друг с другом и со внешней средой
23.  Внутренняя энергия газа. Работа газа.
	1. Внутренняя энергия - энергия тела за вычетом кинетической энергии тела как целого и потенциальной энергии тела во внешнем поле
		1. Кинетическая энергия теплового движения молекул

		3. Внутримолекулярная энергия

	3. Работа газа$$A = \int_{1}^2pdV$$
		1. Если система переходит из состояния 1 в состояние 2 разными путями, то работа будет совершаться различная
		2. Если система совершает круговой процесс(цикл), то работа равна заштрихованной площади
24.  Первое начало термодинамики
	1. Изменить внутреннюю энергию тела можно либо совершив над ним работу $A'$(тогда работа газа $A = -A'$), либо передав ему тепло $Q$
	   Предположим, что внутренняя энергия газа переходит из состояния $U_{1}$ в $U_{2}$
	   Тогда $$U_{2} - U_{1} = A' + Q = Q - A$$$$\Delta U = Q - A \implies Q = \Delta U + A$$
	2. Первое начало термодинамики:$$Q = \Delta U + A$$
	   Количество тепла, сообщаемое системе, тратится на приращение внутренней энергии системы и на совершение этой системой работы над внешними телами
25.  Теплоемкость газа. Уравнение Майера
	1. Теплоёмкость - количество тепла, которое нужно сообщить телу, чтобы нагреть его на 1 K
		1. Удельная - на единицу массы
		2. Молярная - на 1 моль
	2. При $V = const$$$A = 0; \  dQ = dU_{M} = \frac{i}{2}RdT$$$$C_{V} = \frac{dQ_{V}}{dT} = \frac{idT}{2dT}R = \frac{i}{2}R$$
	3. При $p = const$$$A = \int^2_{1}pdV = pd V; \ dQ_{p} = dU_{M} + pd V = \frac{i}{2}RdT + pd V$$$$C_{p} = \frac{dQ_{p}}{dT} = \frac{dU_{M}}{dT} + p\left( \frac{dV_{M}}{dT} \right)_{p}$$$$pV_{M} = RT(\text{из ур-я Менделеева-Клапейрона})\implies V_{M} = \frac{RT}{p}$$
	   $$C_{p} = \frac{dU_{M}}{dT} + p\left( \frac{RdT}{pdT} \right)_{p} = \frac{dU_{M}}{dT} + p \frac{R}{p} = C_{V} + R$$
	4. Уравнение Майера$$C_{p} = C_{V} + R$$
	5. $$C_{V} = \frac{i}{2}R$$$$C_{p} = \frac{i}{2}R + R = R\left( \frac{i}{2} + 1 \right) = R \frac{i+2}{2}$$$$\gamma = \frac{C_{p}}{C_{v}} = \frac{R(i+2)}{2} \frac{2}{iR} = \frac{i+2}{i}$$
26.  Адиабатический процесс. Уравнение адиабаты идеального газа
	1. Адиабатический процесс - процесс, при котором не происходит теплообмена газа с окружающей средой, т.е. $dQ = 0$$$dQ = dU + pdV$$$$\begin{cases}
	    & dU = \frac{m}{M} C_{V} dT \\
		& pdV = \frac{mRT}{M} \frac{dV}{V}
	   
	   \end{cases}$$$$dQ = \frac{m}{M}C_{V}dT + \frac{m}{M}RT \frac{dV}{V} = 0$$
	   $$C_{V} dT + RT \frac{dV}{V} = 0 |:(C_{V}T)$$$$\frac{dT}{T} + \frac{R}{C_{V}} \frac{dV}{V} = 0$$$$d\left( \ln T + \frac{R}{C_{V}} \ln V \right) = 0 \implies \ln T + \frac{R}{C_{V}} \ln V = const$$
	   $$\frac{R}{C_{V}} = \frac{C_{p}-C_{V}}{C_{V}} = \gamma - 1$$
	   2. Уравнение Пуассона$$pV^\gamma = const$$где $\gamma$ - показатель адиабаты
	   
27.  Применение первого начала термодинамики к изопроцессам.
	1. Изопроцесс - процесс, протекающий при каком-либо постоянном параметре состояния системы
	2. Изохорный$$V = const \implies A = 0$$$$dQ = dU$$$$Q_{12} = \Delta U = U_{2} - U_{1} = \frac{m}{M}C_{V}(T_{2}-T_{1})$$$$A_{12} = 0$$$$\frac{p_{1}}{T_{1}} = \frac{p_{2}}{T_{2}} \implies \frac{p_{1}}{p_{2}} = \frac{T_{1}}{T_{2}}$$
	3. Изобарный$$p = const$$$$dQ = dU + pdV$$$$dU = \frac{m}{M} \frac{i}{2} R(T_{2} - T_{1})$$$$pdV = pV_{2} - pV_{1} = \frac{m}{M}R(T_{2} - T_{1})$$$$Q_{12} = \frac{m}{M} \frac{i}{2}R(T_{2} - T_{1}) + \frac{m}{M}R(T_{2} - T_{1}) = \frac{m}{M} R \Delta T \left( \frac{i}{2} + 1 \right) = \frac{m}{M}R\Delta T \frac{i+2}{2} = \frac{m}{M}C_{p}\Delta T$$$$\frac{V_{1}}{V_{2}} = \frac{T_{1}}{T_{2}}$$
	4. Изотермический процесс$$T = const$$$$dU = 0$$$$dQ = dA = pdV$$$$Q = \int^{V_{2}}_{V_{1}} pdV = \int^{V_{2}}_{V_{1}} \frac{m}{M} \frac{RT}{V}dV = \frac{m}{M}RT \int^{V_{2}}_{V_{1}} \frac{dV}{V} = \frac{m}{M}RT \ln \frac{V_{2}}{V_{1}}$$$$p_{1}V_{1} = p_{2}V_{2}$$
	5. Адиабатный процесс$$dQ = 0$$$$U_{12} = \frac{i}{2} \frac{m}{M} R(T_{2} - T_{1})$$$$A_{12} = \int^{V_{2}}_{V_{1}}pdV = \frac{m}{M} \frac{i}{2} R\Delta T = \frac{m}{M}R \Delta T \frac{1}{\gamma-1} =  \frac{p_{2}V_{2} - p_{1}V_{1}}{\gamma - 1}$$
28. Обратимые и необратимые процессы. Круговые процессы (циклы). Тепловая машина.
	1. Круговые процессы - процессы, при которых система, пройдя через несколько промежуточных состояний, возвращается в исходное состояние
	2. Согласно первому началу термодинамики, для того, чтобы газ совершил работу, нужно сообщить ему эквивалентное количество тепла и наоборот, для того, чтобы над газом была совершена работа, нужно отнять эквивалентное количество тепла, такой циклический процесс используется в тепловых машинах, предназначенных для перевода тепловой энергии в механическую
	3. Принцип Томпсона(Кельвина)
	   Невозможно совершить циклический процесс, единственным результатом которого было бы превращение теплоты одного тела в механическую работу другого без каких-либо изменений в других телах
	   Согласно принципу Томпсона в круговом процессе должно участвовать 3 тела: нагреватель $Q_{1}$, рабочее тело $A$ и холодильник $Q_{2}$
	4. КПД тепловой машины$$\eta = \frac{A}{Q_{1}} = \frac{Q_{1} - Q_{2}}{Q_{1}} = 1 - \frac{Q_{2}}{Q_{1}} < 1$$
29. Цикл Карно. Коэффициент полезного действия цикла Карно.
    Рассмотрим наиболее эффективный круговой процесс, в котором участвуют нагреватель, рабочее тело и холодильник
	1. 1-2: Тело изотермически расширяется, при этом работа $A_{1} = RT_{1} \ln \frac{V_{2}}{V_{1}} = Q_{1}$
	2. 2-3: Тело адиабатно расширяется до температуры $T_{2}$
	3. 3-4: Тело изотермически сжимается, при этом $A_{3} = RT_{1}\ln \frac{V_{4}}{V_{3}} = -RT_{2}\ln \frac{V_{3}}{V_{4}} = Q_{2}$
	   Работа, совершаемая в этом цикле, равна площади между кривыми
	4. КПД цикла Карно$$\eta = \frac{A}{Q_{1}} = q - \frac{|Q_{2}|}{Q_{1}} = 1 - \frac{T_{2}}{T_{1}}<0$$
	5. Теоремы Карно
		1. Тепловая машина, работающая при определённых температурах нагревателя и холодильника, не может иметь КПД выше, чем у цикла Карно
		2. КПД цикла Карно зависит только от температур холодильника и нагревателя и не зависит от рабочего тела
30. Второе начало термодинамики.
	1. Формулировка Томпсона(Кельвина)
	   Невозможны такие процессы, единственным результатом которых являлось бы отнятие тепла от одного тела и превращение его полностью в работу
	2. Формулировка Клаузиуса
	   Невозможен процесс, единственным результатом которого была бы передача тепла от менее нагретого тела к более нагретому
	3. Прост формулировка
	   Невозможно создать вечный двигатель 2 порядка, т.е. такую тепловую машину, которая бы отнимала тепло от одного тела и превращала бы его полностью в работу
31. Энтропия в термодинамике.
	1. Любое изменение в системе можно представить как бесконечное количество малых изменений, каждое из которых сопровождается изменением теплоты системы
	   Если система в результате каких-либо изменений переходит обратимым путём из одного состояния в другое, то приведённое количество тепла не зависит от пути, по которому изменялась система$$\oint \frac{\delta Q}{T} = 0\text{ - равенство Клаузиуса}$$
	2. Дифференциальное определение энтропии$$dS = \frac{\delta Q}{T}$$
	3. Интегральное определение$$S = \int^2_{1} \frac{\delta Q}{T}$$
	4. Формулировка I начала термодинамики
	   Возможны только такие процессы, при которых энтропия остаётся неизменной, либо возрастает, при этом необязательно, чтобы энтропия возрастала для каждого тела, участвующего в процессе, увеличивается или остаётся неизменной общая энтропия тел
	5. Свойства энтропии
		1. Энтропия замкнутой системы не изменяется при любом обратимом процессе
		2. Для необратимых процессов изменение энтропии всегда положительно
32. Электрический заряд. Закон сохранения электрического заряда. Закон Кулона.
	1. Существует 2 вида электрических зарядов: положительные и отрицательные
	2. Заряд дискретен, т.е. $q = ne$
	3. Закон сохранения электрического заряда
	   Общий заряд системы остаётся неизменным при любых процессах, происходящих в ней
	4. Закон Кулона$$F_{Кл} = k \frac{|q_{1}||q_{2}|}{r^2}$$
33. Электростатическое поле в вакууме. Напряженность поля. Принцип суперпозиции полей.
	1. Электростатическое поле - особая форма материи, возникающая вокруг неподвижных электрических зарядов. Через электростатическое поле неподвижные заряды взаимодействуют друг с другом
	2. Напряжённость - силовая характеристика поля$$E = \frac{F}{q}$$
	3. Принцип суперпозиции полей
	   Общая напряжённость поля равна сумме напряжённостей полей, создаваемых всеми зарядами системы$$E = \sum E_{i}$$
34. Поле диполя. Диполь во внешнем поле.
	1. Диполь - система из двух частиц, имеющих равные по модулю и противоположные по знаку заряды
	2. Основная характеристика диполя - дипольный момент$$\vec{p} = ql$$
	3. При помещении диполя во внешнее поле на оба заряда диполя действуют силы, равные по модулю и противоположные по знаку, таким образом, результирующая сила, действующая на диполь, помещённый во внешнее поле, равна 0
	4. При помещении диполя во внешнее поле возникает момент силы $M$, стремящейся повернуть диполь так, чтобы вектор дипольного момента был сонаправлен с вектором напряжённости поля$$M = [\vec{p}\vec{E}]$$ 
35. Поток вектора напряженности. Теорема Гаусса для электростатического поля в вакууме.
	1. Вычислим поток вектора напряжённости, проходящего через сферическую поверхность $S$ $$d\vec{S} = dS\vec{n}$$$$\varPhi = \oint \vec{E}d\vec{S} = \oint EdS = \frac{1}{4\pi \epsilon_{0}} \frac{q}{r^2} 4\pi r^2 = \frac{q}{\epsilon_{0}}$$Если линии напряжённости проходят через замкнутую плоскость, то поток вектора напряжённости равен сумме зарядов, лежащих в этой плоскости, делённой на электрическую постоянную
	   Результат справедлив для любой поверхности, т.к. количество силовых линий одинаково
	2. Если количество пересечений силовых линий с замкнутым контуром чётное, то оно эквивалентно 0
	   Если нечётное, то эквивалентно 1
	3. Поток напряжённости электрического поля, создаваемого зарядом, лежащим вне замкнутой поверхности, равен 0
	4. Теорема Гаусса
	   Поток вектора напряжённости в вакууме через замкнутую поверхность числено равен алгебраической сумме зарядов, заключённых в этой плоскости, делённой на электрическую постоянную$$\varPhi = \frac{\sum q_{i}}{\varepsilon_{0}}$$
	   Если заряд распределён в объёме непрерывно, то вводится понятие объёмной плотности заряда $\rho$$$\rho = \frac{dq}{dV} \implies q = \int \rho dV$$$$\varPhi = \frac{1}{\varepsilon_{0}}\int_{V} \rho dV$$теорема Гаусса в интегральной форме
36. Вычисления напряженности поля электростатических систем по теореме Гаусса.
	1. Поле равномерно заряженной бесконечной плоскости
	   Для того, чтобы посчитать заряд, вводится понятие поверхностной плотности заряда($\sigma = \frac{dq}{dS}$)
	   Если $\sigma = const$, то $q = \sigma S$$$2ES = \frac{\sigma S}{\varepsilon_{0}} \implies E = \frac{\sigma}{2\varepsilon_{0}}$$
	2. Поле двух равномерно заряженных бесконечных плоскостей$$ES = \frac{\sigma S}{\varepsilon_{0}} = \frac{\sigma}{\varepsilon_{0}}$$
	3. Поле равномерно заряженной сферы
		1. $r\geq R$$$E4\pi r^2 = \frac{q}{\varepsilon_{0}} \implies E = \frac{q}{4\pi r^2\varepsilon_{0}}$$
		2. $r<R$$$E = 0$$
	4. Поле равномерно заряженного шара
		1. $r\geq R$
		   То же самое, что и у сфера
		2. $r<R$$$E = \frac{qr}{4\pi\varepsilon_{0}R^3}$$
	5. Поле равномерно заряженного цилиндра
	   $\tau = \frac{dq}{dl}$ - линейная плотность заряда
		1. $r\geq R$$$E = \frac{\tau}{2\pi r\varepsilon_{0}}$$
		2. $r < R$$$E = 0$$
37. Работа сил электростатического поля. Потенциал. Теорема о циркуляции вектора напряженности.
	1. $$A = \int_{r_{1}}^{r_{2}} Fdr =\int_{r_{1}}^{r_{2}} \frac{1}{4\pi\varepsilon_{0}} \frac{|q_{1}||q_{2}|}{r^2} = \frac{|q_{1}||q_{2}|}{4\pi\varepsilon_{0}} \int_{r_{1}}^{r_{2}} \frac{1}{r^2} = \frac{|q_{1}||q_{2}|}{4\pi\varepsilon_{0}} \left( \left. -\frac{1}{r} \right\vert^{r_{2}}_{r_{1}} \right) = \frac{|q_{1}||q_{2}|}{4\pi\varepsilon_{0}}\left( \frac{1}{r_{1}} - \frac{1}{r_{2}} \right)$$
	2. Потенциал - физическая величина, числено равная отношению потенциальной энергии к заряду$$\phi = \frac{U}{q_{0}} \implies U = \varphi q_{0}$$$$A = U_{1} - U_{2} = q_{0}(\varphi_{1} - \varphi_{2})\implies \varphi_{1} - \varphi_{2} = \frac{A}{q_{0}}$$
	3. Теорема о циркуляции вектора напряжённости
	   Если траектория движения заряда - замкнутый контур, то работа внешних сил по перемещению этого заряда $A = \int Fdr = 0$
	   Следовательно, $\oint_{l}Edl = 0$
	   Значит, линии напряжённости электростатического поля не могут быть разомкнуты и заканчиваются/начинаются только на зарядах
38. Связь между напряженностью и потенциалом$$\vec{E} = -grad \varphi$$
39. Поляризация диэлектриков. Напряженность поля в диэлектриках
	1. Диэлектрики - вещества, в которых отсутствуют свободные заряды, следовательно, не способные проводить электрический ток
	2. Виды диэлектриков
		1. Полярные
		   Молекулы которых ассиметричны
		2. Неполярные
		   Молекулы которых симметричны
		3. Ионные
		   Вещества, обладающие двумя ионными подрешётками
	3. Виды поляризации
		1. Ориентационная - в электростатическом поле молекулы полярных диэлектриков поворачиваются под воздействием момента силы, стремящегося повернуть их так, чтобы вектор дипольного момента был сонаправлен с вектором напряжённости поля
		2. Электронная - в электростатическом поле электронные оболчки неполярных диэлектриков деформируются, из-за чего возникает дипольный момент
		3. Ионная - смещение электронных подрешёток вызывает дипольный момент
	4. Для описания поляризации вводится физическая величина - поляризованность, отношение дипольного момента к объёму вещества$$\vec{p} = \frac{\vec{p_{V}}}{V}$$
	   Экспериментально установлено, что поляризованность связана с напряжённостью поля следующим соотношением$$\vec{E} = \chi \vec{p}\varepsilon_{0}$$где $\chi$ - диэлектрическая восприимчивость
	5. Напряжённость поля в диэлекриках
	   Для описания напряжённости поля в диэлектрике рассмотрим плоскую диэлектрическую заряженную пластину пластину
	   На её поверхности находятся заряды $\sigma'$, называемые связанными
	   Внутри диэлектрика есть электростатическое поле с напряжённостью $E'$, а снаружи - с напряжённостью $E_{0}$
	   Тогда $E = \vec{E'} + \vec{E_{0}} = E_{0} - E'$
	   $E' = \frac{\sigma'}{\varepsilon_{0}}$
	   $E = E_{0} - \frac{\sigma'}{\varepsilon_{0}}$
	   Вычислим величину $\sigma'$$$\begin{cases} && p_{V} = pV = pSd \\
	   && p_{V} = ql = \sigma'Sd
	   \end{cases}$$$$p = \sigma'$$
	   С другой стороны, $p = \chi \varepsilon_{0} E$$$E = E_{0} - E' = E_{0} - \frac{\sigma'}{\varepsilon_{0}} = E_{0} - \frac{p}{\varepsilon_{0}} = E_{0} - \chi E$$$$E = \frac{E_{0}}{1 + \chi} = \frac{E_{0}}{\varepsilon}$$
40. Вектор электрического смещения. Теорема Гаусса для электрического смещения.
	1. Вектор электрического смещения - характеристика электростатического поля, созданного свободными зарядами при таком их распределении в пространстве, которое имеется при наличии диэлектрика$$\vec{D} = \varepsilon_{0}\varepsilon \vec{E} = \varepsilon_{0}\vec{E}(1+\chi) = \varepsilon_{0}\vec{E} + \varepsilon_{0}\vec{E}\chi = \varepsilon_{0}\vec{E} + \vec{p}$$
	2. Вектор электрического смещения отображается так же, как и вектор напряжённости поля, но может начинаться и заканчиваться только на свободных зарядах, в отличие от вектора напряжённости поля, который может начинаться и заканчиваться как на свободных, так и на связанных зарядах
	3. Теорема Гаусса для электрического смещения$$\varPhi_{D} = \oint_{S} \vec{D}d\vec{S} = \oint_{S} D_{n}dS = \sum_{i=1}^{N} Q_{i}$$где $Q_{i}$ - свободные заряды
	   В вакууме($\varepsilon = 1$)$$D_{n} = \varepsilon_{0}E_{n}$$$$\varPhi_{D} = \oint_{S}D_{n}dS = \oint_{S}\varepsilon_{0}E_{n}dS = \varepsilon_{0}\oint E_{n}dS = \sum_{i=1}^{N} Q_{i}$$$$\varPhi_{E} = \oint_{S}E_{n}dS = \frac{\sum Q_{i}}{\varepsilon_{0}}$$
	   Поток вектора электрического смещения $\vec{D}$ при прохождении через замкнутую поверхность числено равен сумме свободных зарядов, лежащих на этой поверхности
41. Проводники в электростатическом поле.
	1. Если поместить проводник во внешнее поле, то заряды внутри проводника будут перемещаться до тех пор, пока вектор напряжённости поля внутри проводника не станет равным 0
	2. Если сообщить проводнику избыточный заряд, то он распределится по поверхности проводника
	   По теореме Гаусса $$\oint_{S}DdS = \sum q_{i}$$$$DS = q = \sigma S$$$$E = \frac{D}{\varepsilon_{0}\varepsilon} = \frac{\sigma}{\varepsilon_{0}\varepsilon}$$
	3. Если в поле внести нейтральный проводник, то заряды внутри него будут перераспределяться. Явление перераспределения зарядов в проводнике называют индукцией, а перераспределённые заряды называют индуцированными. Индуцированные заряды располагаются на поверхности проводника
42. Конденсаторы. Электроемкость конденсаторов. Соединения конденсаторов.
	1. Конденсатор - устройство, состоящее из двух разноимённо заряженных проводников и диэлектрика между ними, способное накапливать электричество
	2. Электроёмкость$$C = \frac{q}{\varphi}$$
	3. Примеры электроёмкости
		1. Плоский конденсатор$$C = \frac{\varepsilon \varepsilon_{0}S}{d}$$
		2. Сферический конденсатор$$C = 4\pi\varepsilon\varepsilon_{0} \frac{r_{1}r_{2}}{r_{1} - r_{2}}$$
		3. Цилиндрический конденсатор$$C = \frac{2\pi\varepsilon\varepsilon_{0}l}{\ln \frac{r_{1}}{r_{2}}}$$
	4. Соединения конденсаторов
		1. Параллельное$$C = \sum C_{i}$$
		2. Последовательное$$\frac{1}{C} = \sum \frac{1}{C_{i}}$$
43. Энергия взаимодействия системы зарядов. Энергия конденсатора и заряженного проводника. Энергия электрического поля.
	1. Энергия взаимодействия системы зарядов$$W = \frac{1}{2}\sum q_{i}\varphi_{i}$$
	2. Конденсатора$$W = \frac{q\Delta \varphi}{2}$$где $\Delta \varphi$ - разность потенциалов на между обкладками конденсатора
	3. Заряженного проводника$$W = \frac{q\varphi}{2}$$
	4. Электростатического поля$$w = \frac{W}{V} = \frac{\varepsilon\varepsilon_{0}E^2}{2} = \frac{D^2}{2\varepsilon\varepsilon_{0}}$$
44. Электрический ток. Сила тока и плотность тока.
	1. Электрический ток - направленное движение заряженных частиц
	2. Носители тока - заряженные частицы, участвующие в направленном движении
	3. Для возникновения электрического тока необходимо 2 условия
		1. Наличие свободных зарядов
		2. Наличие электрического поля внутри проводника
	4. Сила тока$$I = \frac{dq}{dt}$$
	   Постоянный ток - ток, характеристики которого не меняются со временем($q = const, \ t = const \implies I = \frac{q}{t} = const$)
	5. Если ток распределён неравномерно внутри проводника, то вводится микроскопическая характеристика - плотность тока
		   Плотность тока - вектор, модуль которого равен отношению заряда, прошедшего через малую перпендикулярную току площадку $dS_{\perp}$ за интервал времени $dt$$$\vec{j} = \frac{dq}{dS_{\perp}dt} = \frac{dI}{dS_{\perp}}$$
45. ЭДС
	1. Электродвижущая сила - работа суммы неэлектрических сил по перемещению заряда$$\varepsilon = \frac{A_{ст}}{q} = \int_{1}^2  \vec{E_{ст}}\, d\vec{l}$$
	2. Падение напряжения
		1. На однородном участке цепи$$U_{12} = \varphi_{1} - \varphi_{2}$$
		2. На неоднородном участке цепи$$U_{12} = \varepsilon_{12} + (\varphi_{1} - \varphi_{2})$$
46. Закон Ома для однородного участка цепи (интегральная и дифференциальная форма).
	1. Интегральная форма$$I = \frac{U}{R}$$где $U = \varphi_{1} - \varphi_{2}$, а $R = \rho \frac{l}{S}$
	2. Дифференциальная форма$$\begin{flalign} \\
	& dR = \rho \frac{dl}{dS} \\
	& dU = \varphi_{1} - \varphi_{2} = \vec{E}d\vec{l} \\
	& dI = \frac{dU}{dR} = \frac{\vec{E}d\vec{l}dS}{\rho dl} = \frac{\vec{E}dS}{\rho} = \vec{j}dS
	   \end{flalign}$$$$j = \frac{1}{\rho}E = \sigma E$$где $\sigma$ - удельная электропроводность
47. Закон Ома для неоднородного участка цепи (интегральная и дифференциальная форма).
	1. Дифференциальная форма$$\vec{j} = q(\vec{E} + \vec{E_{стор}})$$
	2. Интегральная форма$$I = \frac{\varepsilon_{12} + (\varphi_{1} - \varphi_{2})}{R + r}$$Если цепь замкнута, то $\varphi_{1} = \varphi_{2}$, тогда $$I = \frac{\varepsilon_{12}}{R+r}$$
48. Мощность тока. КПД источника тока. 
	1. $$P = \frac{A}{t} = \frac{UIt}{t} = UI$$
	   Если цепь замкнута, то полная мощность, выделяемая на цепи$$P = I\varepsilon$$
	   Полезная мощность$$P_{п} = UI = I^2R$$
	2. КПД источника тока$$\eta = \frac{P_{п}}{P} = \frac{I^2R}{I\varepsilon} = \frac{IR}{\varepsilon} = \frac{\varepsilon R}{(R+r)\varepsilon} = \frac{R}{R+r}$$
49. Закон Джоуля - Ленца в интегральной и дифференциальной формах.
	1. Интегральная форма$$Q = UIt = I^2Rt$$
	2. Дифференциальная форма
	   Введём понятие удельной тепловой мощности$$Q_{уд} = \frac{dQ}{dtdV}$$$$\begin{flalign} & I = jS \\ 
	   & dR = \frac{\rho dl}{S} = \frac{1}{\sigma} \frac{dl}{S} \\ 
	   & dV = Sdl
	   \end{flalign}$$$$dQ = I^2dRdt = j^2S^2 \frac{1}{\sigma} \frac{dl}{S} dt = j^2 \frac{dVdt}{\sigma}$$$$\frac{dQ}{dVdt} = \frac{j^2}{\sigma} \implies \frac{j^2}{\sigma} = Q_{уд}$$
50. Разветвленные цепи. Правила Кирхгофа.
	1. Узел - точка, в которой сходятся 3 и более участка цепи
	2. Правила Кирхгофа
		1. Сумма токов в узлах цепи равна 0$$\sum_{k} I_{k} = 0$$
		2. В замкнутых контурах цепи сумма падений тока равна сумме произведений силы тока на сопротивление данного участка цепи$$\sum_{k} \varepsilon_{k} = \sum_{k} I_{k}R_{k}$$
	3. Алгоритм нахождения тока на любом участке цепи, используя правила Кирхгофа
		1. Произвольно выбрать направление тока на всех участках цепи. Если в конце вычислений $I>0$, то выбранное направление тока совпадает с реальным
		2. Найти все узлы в цепи и составить для каждого уравнение по I правилу Кирхгофа
		3. Найти все замкнутые контуры в цепи, выбрать направление обхода для каждого из них и составить для уравнения по II правилу Кирхгофа
		4. Занести полученные уравнения в систему
		5. Убедиться, что в полученной системе уравнений есть ЭДС и сопротивления всех участков цепи
		6. Решить полученную систему уравнений
		7. Хоба!
