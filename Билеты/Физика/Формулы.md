1. Механика
	1. Кинематика
		1. $x = x_{0} + v_{0}t + \frac{at^2}{2}$
		2. $v = v_{0} + at$
		3. $\vec{v} = \frac{d\vec{r}}{dt}$
		4. $v = \sqrt{ v^2_{x} + v^2_{y} + v^2_{z} }$
		5. $\vec{a} = \frac{d\vec{v}}{dt} = \frac{d^2\vec{r}}{dt^2}$
		6. $a = \sqrt{ a^2_{n} + a^2_{\tau} }$
		7. $\vec{a_{\tau}} = \frac{d\vec{v}}{dt} = \epsilon R$
		8. $\vec{a_{n}} = \frac{\vec{v}^2}{R^2} = \omega^2R$
		9. $<\vec{a}> = \frac{\Delta \vec{v}}{\Delta t}$
		10. $\vec{a_{момент}} = \lim_{ \Delta t \to 0 } \frac{\Delta \vec{v}}{\Delta t} = \frac{d\vec{v}}{dt}$
		11. $\omega = \frac{d\varphi}{dt}$
		12. $\varepsilon = \frac{d\omega}{dt} = \frac{d^2\varphi}{dt^2}$
		13. $\varphi = \varphi_{0} + \omega_{0}t + \frac{\varepsilon t^2}{2}$
		14. $\omega = \omega_{0} + \varepsilon t$
	2. Динамика
		1. $\vec{F} = m\vec{a} = m \frac{d\vec{v}}{dt} = \frac{dm\vec{v}}{dt} = \frac{d\vec{p}}{dt}$
		2. $\vec{p} = m\vec{v}$
		3. $dA = Fdr = FdS$
		4. $E_{k} = \frac{mv^2}{2}$
		5. $E_{p} = mgh = -G \frac{mM}{R+r} = \frac{k\Delta x^2}{2}$
		6. $\vec{M} = [\vec{r}\vec{F}]$
		7. $\vec{L} = [\vec{r}\vec{p}]$
		8. $I = \int r^2dm$
