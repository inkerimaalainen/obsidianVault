1. **Электрический заряд. Закон сохранения электрического заряда. Закон Кулона**
	1. Электрический заряд
	   2. Установлено, что электрический заряд бывает ==отрицательным== и ==положительным==
	   3. Носителями зарядов являются ==элементарные частицы==, заряд почти всех элементарных частиц - либо 0, либо равен элементарному заряду$$e = 1,6 * 10^{-19} Кл$$
	   4. Электрический заряд тела дискретен$$q = \pm ne$$
	   5. Электрический заряд измеряется в Кулонах(Кл)
	   6. ==Точечный заряд== - заряженное тело, размерами которого можно пренебречь по сравнению с расстояниями других тел
	7. ==Закон сохранения электрического заряда==
	   8. Алгебраическая сумма зарядов в замкнутой системе не изменяется при любых процессах внутри системы
	9. ==Закон Кулона(закон взаимодействия точечных зарядов)==
	   10. Сила взаимодействия двух неподвижных точечных зарядов, находящихся в вакууме, пропорциональна величинам зарядов $q_{1}$ и $q_{2}$ и обратно пропорциональна квадрату расстояния $r$ между ними$$F_{Кл} = k \frac{q_{1}q_{2}}{r^2}$$где $k$ - коэффициент пропорциональности
	   
	   11. Сила направлена вдоль проходящей через заряды прямой$$k = \frac{1}{4\pi \epsilon_{0}} \implies \vec{F} = \frac{1}{4\pi \epsilon_{0}} * \frac{q_{1}q_{2}}{r^3}\vec{r}$$где $\epsilon_{0} = 8,85 * 10^{-12} \frac{Ф}{м}$ - электрическая постоянная
2. **Электростатическое поле в вакууме. Напряженность поля. Принцип суперпозиции полей**
	1. ==Электростатическое поле== - поле, создаваемое неподвижными зарядами
	   Для обнаружения и исследования поля используется ==пробный заряд==(точечный положительный заряд)
	2. ==Напряжённость поля== - сила, действующая на единичный положительный заряд, помещённый в данную точку пространства$$\vec{E} = \frac{1}{4\pi \epsilon_{0}} * \frac{q}{r^3}\vec{r}$$
	   Графически электростатические поля обозначаются с помощью силовых линий. Линии вектора $\vec{E}$ строятся таким образом, чтобы касательная в каждой точке совпадала с направлением вектора напряжённости. Вектор напряжённости ==всегда направлен от + к -==. Густота силовых линий характеризуется модулем вектора напряжённости
	3. ==Однородное поле== - поле, напряжённость которого не изменяется$$\vec{E} = const$$
	4. ==Поток вектора напряжённости==$$d \vec{S} = d S \vec{n}$$$$d Ф_{E} = \vec{E} d \vec{S} = EdS\cos \alpha = E_{n} dS$$Полный поток:$$Ф_{E} = \oint_S \vec{E} \, d \vec{S} = \oint_{S} E_{n} dS$$
	5. ==Принцип суперпозиции полей==$$\vec{\sum{F_{i}}} = \sum q_{0} \vec{E_{i}}$$$$q_{0} \vec{E} = \sum \vec{E_{i}}q_{0}$$$$\vec{E} = \sum \vec{E_{i}}$$
	   Напряжённость поля, создаваемого системой зарядов в данной точке пространства равна векторной сумме напряжённости полей, создаваемых каждым зарядом

4. **Поток вектора напряженности. Теорема Гаусса для электростатического поля в вакууме**
	1. ![[Pasted image 20260519232650.png]]$$d \vec{S} = dS \vec{n}$$$$Ф_{E} = \oint_{S} \vec{E} d \vec{S} = \oint EdS = \frac{1}{4\pi \epsilon_{0}r^2} \frac{q}{r^2}*4\pi r^2 = \frac{q}{\epsilon_{0}}$$Результат справедлив для любой поверхности, т.к. количество силовых линий вектора $\vec{E}$ одинаково
	2. Если замкнутая поверхность произвольной формы охватывает заряды, то число силовых линий с учётом знака:
		- если число пересечений ==нечётно==, то эквивалентно 1
		- если ==чётно== - 0
		Поток электрического поля, создаваемого зарядом, находящимся вне замкнутой поверхности, через эту поверхность равен 0
	3. ==Теорема Гаусса== в интегральной форме для системы зарядов$$Ф_{E} = \oint \vec{E} d \vec{S} = \frac{\sum{q_{i}}}{\epsilon_{0}}$$
	4. ==Теорема Гаусса==
	   Поток вектора напряжённости $\vec{E}$ в вакууме через замкнутую поверхность численно равен алгебраической сумме зарядов, заключённых внутри поверхности, делённой на $\epsilon_{0}$(для дискретных зарядов)
	   Если заряд распределён в объёме непрерывно, то вводится понятие объёмной плотности заряда $\rho$$$Ф_{E} = \oint_{V}\rho dV$$

6. **Работа сил электростатического поля. Потенциал. Теорема о циркуляции вектора напряженности**
	1. ![[Pasted image 20260520231820.png]]Заряд $q_0$ перемещается из точки 1 в точку 2 в поле заряда $q$. При этом сила совершает работу:$$dA = \vec{F}d \vec{l}$$где $d \vec{l}$ - элементарное перемещение
	   Из [[Pasted image 20260520231820.png | рисунка]]:$$dA = \vec{F} d \vec{l} = Fdl\cos \alpha = Fdr = \frac{q_{0}q}{4 \pi \epsilon_{0}} \frac{dr}{r^2}$$$$A = \int_{1}^2dA = \int_{1}^2\frac{q_{0}q}{4 \pi \epsilon_{0}} \frac{dr}{r^2} = \frac{q_{0}q}{4 \pi \epsilon_{0}} \int_{1}^2 \frac{dr}{r^2} = \frac{q_{0}q}{4 \pi \epsilon_{0}} \left( \frac{1}{r_{1}} - \frac{1}{r_{2}}\right)$$==Работа не зависит от вида траектории, а определяется только начальным и конечным положениями заряда
	   Такое поле называется <u>потенциальным</u>, а действующие в нём силы - <u>потенциальными</u>(консервативными)==
	   В электростатическом поле:$$\oint dA = 0; \ (r_{1} = r_{2})$$
	   Если тело находится в потенциальном поле, то оно обладает потенциальной энергией
	   ==Работа потенциальных сил== осуществляется за счёт убыли потенциальной энергии$$A_{12} = \frac{q_{0}q}{4 \pi \epsilon_{0}}\left( \frac{q}{r_{1}} - \frac{q}{r_{2}} \right) = U_{1} - U_{2}$$где U - потенциальная энергия
	   
	2. Теорема о циркуляции $\vec{E}$
	   Поэтому для электростатического поля можно записать: $$\oint_{L} \vec{E} d \vec{l} = 0$$Интеграл такого вида называется циркуляцией $\vec{E}$
	   Из этого следует, что силовые линии незамкнуты, они начинаются и заканчиваются на зарядах или одним концом уходят в бесконечность
	3. ==Потенциальная энергия(U)== точечного заряда $q_{0}$ в поле заряда $q$ - физическая величина вида$$U = \frac{1}{4 \pi \epsilon_{0}} \frac{q_{0}q}{r}$$
	   Потенциальная энергия - алгебраическая величина
	   Для n-зарядов$$U = \sum U_{1}$$
	4. ==Потенциал электростатического поля== - физическая величина, равная отношению потенциальной энергии к заряду$$\phi = \frac{U}{q_{0}}$$Для точечного заряда:$$\phi = \frac{1}{4 \pi \epsilon_{0}} \frac{q}{r}$$$$A_{12} = U_{1} - U_{2} = q_{0}(\phi_{1} - \phi_{2}) \implies (\phi_{1} - \phi_{2}) = \frac{A_{12}}{q_{0}}$$
	5. ==Разность потенциалов== - работа по перемещению единичного положительного заряда из одной точки поля в другую$$A_{12} = \int_{1}^2 \vec{F} d \vec{l} = \int_{1}^2 q_{0} \vec{E} d \vec{l}$$$$\Delta \phi = \int_{1}^2 \vec{E} d \vec{l}$$$Если перемещать заряд $q_{0}$ из точки в бесконечность$$\phi = \frac{A_{\infty}}{q_{0}}$$т.е. потенциал данной точке поля равен работе по перемещению единичного положительного заряда из данной точки поля в бесконечность
	   Потенциал измеряется в вольтах $\text{В = } \frac{\text{Дж}}{\text{Кл}}$
