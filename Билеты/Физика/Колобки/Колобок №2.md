1. **Электрический заряд. Закон сохранения электрического заряда. Закон Кулона**
	1. Электрический заряд
	   2. Установлено, что электрический заряд бывает ==отрицательным== и ==положительным==
	   3. Носителями зарядов являются ==элементарные частицы==, заряд почти всех элементарных частиц - либо 0, либо равен элементарному заряду$$e = 1,6 * 10^{-19} Кл$$
	   4. Электрический заряд тела дискретен$$q = \pm ne$$
	   5. Электрический заряд измеряется в Кулонах(Кл)
	   6. ==Точечный заряд== - заряженное тело, размерами которого можно пренебречь по сравнению с расстояниями других тел
	7. ==Закон сохранения электрического заряда==
	   8. Алгебраическая сумма зарядов в замкнутой системе не изменяется при любых процессах внутри системы
	9. ==Закон Кулона(закон взаимодействия точечных зарядов)==
	   10. Сила взаимодействия двух неподвижных точечных зарядов, находящихся в вакууме, пропорциональна величинам зарядов $q_{1}$ и $q_{2}$ и обратно пропорциональна квадрату расстояния $r$ между ними$$F_{Кл} = k \frac{q_{1}q_{2}}{r^2}$$где $k$ - коэффициент пропорциональности
	   
	   11. Сила направлена вдоль проходящей через заряды прямой$$k = \frac{1}{4\pi \epsilon_{0}} \implies \vec{F} = \frac{q}{4\pi \epsilon_{0}} * \frac{q_{1}q_{2}}{r^3}\vec{r}$$где $\epsilon_{0} = 8,85 * 10^{-12} \frac{Ф}{м}$ - электрическая постоянная
2. **Электростатическое поле в вакууме. Напряженность поля. Принцип суперпозиции полей**
	1. ==Электростатическое поле== - поле, создаваемое неподвижными зарядами
	   Для обнаружения и исследования поля используется ==пробный заряд==(точечный положительный заряд)
	2. ==Напряжённость поля== - сила, действующая на единичный положительный заряд, помещённый в данную точку пространства$$\vec{E} = \frac{1}{4\pi \epsilon_{0}} * \frac{q}{r^3}\vec{r}$$
	   Графически электростатические поля обозначаются с помощью силовых линий. Линии вектора $\vec{E}$ строятся таким образом, чтобы касательная в каждой точке совпадала с направлением вектора напряжённости. Вектор напряжённости ==всегда направлен от + к -==. Густота силовых линий характеризуется модулем вектора напряжённости
	3. ==Однородное поле== - поле, напряжённость которого не изменяется$$\vec{E} = const$$
	4. ==Поток вектора напряжённости==$$d \vec{S} = d S \vec{n}$$$$d Ф_{E} = \vec{E} d \vec{S} = EdS\cos \alpha = E_{n} dS$$Полный поток:$$Ф_{E} = \oint_S \vec{E} \, d \vec{S} = \oint_{S} E_{n} dS$$
	5. ==Принцип суперпозиции полей==$$\vec{\sum{F_{i}}} = \sum q_{0} \vec{E_{i}}$$$$q_{0} \vec{E} = \sum \vec{E_{i}}q_{0}$$$$\vec{E} = \sum \vec{E_{i}}$$
	   Напряжённость поля, создаваемого системой зарядов в данной точке пространства равна векторной сумме напряжённости полей, создаваемых каждым зарядом
